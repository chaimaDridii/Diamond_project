# Boutique Diamond

Bienvenue dans le projet **Boutique Diamond** ! Ce projet est une application web de boutique en ligne permettant aux utilisateurs de consulter des produits, ajouter des articles à leur panier et passer une commande.

## Prérequis

Avant de démarrer le projet, assurez-vous que les prérequis suivants sont installés :

- PHP (version 7.4 ou supérieure)
- Serveur web (XAMPP, WAMP, etc.)
- MySQL
